# Zero-Day Defense System Dashboard (Redesign)

This is a production-grade, enterprise-ready cybersecurity dashboard built with React, Tailwind CSS, and Agentic AI integration concepts.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the interactive development server:
```bash
npm run dev
```
The dashboard will be available at `http://localhost:5173`.

## 🎨 Design Philosophy
- **Tactical Aesthetic**: Deep navy black (#0B0F17) with high-contrast data visualization.
- **Agentic AI First**: Dedicated "AI Insights" panel and "Agent Rationale" for every network event.
- **Data-Driven**: Score distribution across Perception, Forecasting, and LSTM layers.
- **Enterprise Ready**: Modular component architecture, Framer Motion animations, and responsive layout.

## 🧱 Component Architecture
- `App.jsx`: Main dashboard container with live simulation state.
- `Header.jsx`: Top navigation with critical system status and primary CTAs.
- `MetricCard.jsx`: Hero metrics with trend indicators and dynamic indicators.
- `AIInsights.jsx`: Real-time anomaly detection and suggested mitigation triggers.
- `ThreatCharts.jsx`: Advanced score distribution and probability histograms using Recharts.
- `EventAuditTable.jsx`: High-performance data table with expandable rationale and per-layer AI scores.
